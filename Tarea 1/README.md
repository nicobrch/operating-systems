### Ejecución

- Compilar ambos archivos .cpp con g++.

```shell
g++ mesa.cpp -o mesa
g++ jugador.cpp -o jugador
```

- Ejecutar mesa en una terminal e indicar cuántos N jugadores habrán.

```shell
./mesa
```

- Ejecutar N jugadores en N terminales distintas.

```shell
./jugador
```
